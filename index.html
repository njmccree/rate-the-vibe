Connect Firebase

One-time setup · ~3 minutes

ℹ️ Your friends won't need to do this — your share link includes everything automatically.
1
Go to console.firebase.google.com and sign in with Google
2
Tap Add project → any name → skip Analytics → Create project
3
Left menu → Build → Realtime Database → Create database → any region → Start in test mode
4
Copy the URL — looks like https://myapp-default-rtdb.firebaseio.com

iOS Shortcut Setup
Add songs instantly from Apple Music or Shazam.
How to set it up:
1. Open Shortcuts app → tap +
2. Add action: "Shazam: Recognize Music" (or "Get Current Song" from Music)
3. Add "URL Encode" → set to Magic Variable Name → rename EncodedTitle
4. Add "URL Encode" → set to Magic Variable Artist → rename EncodedArtist
5. Add "Open URLs" → paste URL below → replace TITLE→EncodedTitle, ARTIST→EncodedArtist, STOREID→Magic Variable Store ID
